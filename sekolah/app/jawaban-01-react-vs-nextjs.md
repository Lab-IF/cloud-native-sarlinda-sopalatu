## 📝 Kuis Singkat

### 1. Next.js adalah...

**Jawaban:** _(Sebuah Framework React yang bersifat open-source untuk membangun aplikasi web modern. Next.js memungkinkan fitur seperti Server-Side Rendering (SSR) dan pembuatan situs statis (Static Site Generation) secara bawaan untuk meningkatkan performa dan SEO.)_

### 2. Mana yang TIDAK perlu di-install manual di Next.js?

**Jawaban:** _(Routing (Sistem Navigasi). Berbeda dengan React biasa yang membutuhkan library tambahan seperti react-router-dom, Next.js sudah memiliki sistem File-based Routing secara otomatis di dalam folder app atau pages.)_

### 3. Saat install Next.js, agar menggunakan JavaScript (bukan TypeScript), kita harus pilih...

**Jawaban:** _(Pilih opsi "No" pada pertanyaan prompt "Would you like to use TypeScript?" saat menjalankan perintah instalasi npx create-next-app@latest.)_

### 4. Apakah komponen React bisa dipakai di Next.js?

**Jawaban:** _(Bisa. Karena Next.js dibangun di atas React, semua komponen standar React dapat digunakan. Namun, perlu diperhatikan perbedaan antara Client Components (menggunakan direktif 'use client') dan Server Components (bawaan Next.js))_

### 5. Sebutkan minimal 3 fitur yang Next.js berikan di atas React biasa!

**Jawaban:** _(Rendering yang Fleksibel: Mendukung Server-Side Rendering (SSR), Static Site Generation (SSG), dan Incremental Static Regeneration (ISR).

Optimasi Gambar (Image Optimization): Menggunakan komponen <Image /> untuk kompresi dan lazy loading otomatis.

API Routes: Memungkinkan pembuatan endpoint API (Backend) langsung di dalam struktur folder project yang sama tanpa memerlukan server Node.js terpisah.

Fast Refresh: Pengalaman pengembangan yang lebih cepat dengan pembaruan tampilan instan tanpa kehilangan state aplikasi saat kode diubah)_
